遊び方

全員で 1人の冒険者 を動かし、降り階段 を目指す協力ゲームです。

  • 各自「自分の操作（上下左右 / とどまる）」を、他の人に見えない状態で選んで決定します。
  • 全員が決定すると、その階の順番（1番手→2番手→…）で操作が実行されます。順番は設定で「階ごとにランダム / 固定」を選べます。
  • 全操作を終えた時点で冒険者が階段の上にいればクリア。壁や画面外へは進めず「空振り」します。
  • 合計手数 ＞ 最短距離 のときは、誰かが「とどまる」を選ぶ必要があります。
  • 失敗しても3回まで挑戦可能。失敗すると全員の選択が公開されるので、それを読んで次の手を合わせましょう。
  • 封印：ふつう以上では、各自に「使えない方向」が公開で割り振られます。誰がどの方向を担当できるか、全員の封印から推理しましょう。
  • ジェム：寄り道の位置にあるサブ目標。クリア時に全回収で★+1、一発クリアで★+1（最大★3）。全ジェム回収は必ず可能な配置になっています。
  • 階段を降りて進む：クリアするたびに次の階へ。合計★を積み上げて記録を伸ばしましょう。ミスが尽きたら冒険は終了です。
  • ピン（設定でON）：選択中に盤面をタップして仲間に意思表示できます（1人2個まで）。難しすぎるときの補助にどうぞ。

※ 別のタブ / 端末で同じURLを開くと、その分だけ参加できます（2〜6人）。
